# جمعياتبرو - Jamiat Pro 🏛️

> منصة متكاملة لإدارة وتنظيم الجمعيات مع نظام حماية احترافي وتصميم ثلاثي الأبعاد

**المطور:** زعيم الدمار

---

## ✨ المميزات

- 🎨 **تصميم ثلاثي الأبعاد مذهل** - Particles + Glassmorphism + درع أمان متحرك
- 🔐 **نظام حماية متقدم** - تشفير bcrypt + 2FA + سجل أمان
- 🏢 **إدارة الجمعيات** - إنشاء وتعديل وحذف وتصنيف
- 👥 **إدارة الأعضاء** - إضافة أدوار وحالات وتتبع
- 📅 **إدارة الفعاليات** - جدولة وتتبع الميزانيات
- 💰 **الإدارة المالية** - دخل ومصروفات وتقارير
- 📊 **لوحة تحكم ذكية** - رسوم بيانية تفاعلية
- 🛡️ **مركز الأمان** - درجة أمان + سجل أنشطة + 2FA
- 👑 **لوحة تحكم المشرف العام** - إدارة المستخدمين والنظام
- 🔔 **نظام إشعارات** - تنبيهات فورية
- 📤 **تصدير التقارير** - CSV/JSON
- 🌐 **دعم تعدد اللغات** - عربي/إنجليزي

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- npm أو bun
- قاعدة بيانات PostgreSQL (موصى به: [Neon](https://neon.tech))

### الخطوات

```bash
# 1. استنساخ المشروع
git clone https://github.com/hmadhmqlwy-droid/hmadhmqlwy-droid.git
cd hmadhmqlwy-droid

# 2. تثبيت الحزم
npm install

# 3. إعداد متغيرات البيئة
cp .env.example .env
# عدّل ملف .env وأضف رابط قاعدة البيانات

# 4. إنشاء قاعدة البيانات
npx prisma migrate dev --name init

# 5. إنشاء حساب الأدمن
npx tsx prisma/seed.ts

# 6. تشغيل المشروع
npm run dev
```

### بيانات الدخول الافتراضية
- **البريد:** `admin@jamaat.pro`
- **كلمة المرور:** `Admin@2026`

## 🌐 النشر على Vercel

1. أنشئ قاعدة بيانات PostgreSQL مجانية على [Neon](https://neon.tech)
2. اذهب إلى [vercel.com](https://vercel.com) وسجّل بحساب GitHub
3. استورد المشروع من GitHub
4. أضف متغيرات البيئة:
   - `DATABASE_URL` = رابط Neon (مع `?sslmode=require`)
   - `DIRECT_URL` = نفس رابط Neon
5. Build Command: `npm run vercel-build`
6. اضغط Deploy 🚀
